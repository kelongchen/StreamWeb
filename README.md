# video-stream-web
## Description

> This is a stream video website written in `Golang` which let people share interesting videos

## Progress
- [x] Uploading Videos

- [x] Register

## Usage
1. Run `initialdb.sql`
2. Run `build.sh`
3. Run `buildprod.sh`
4. Run `deploy.sh`
